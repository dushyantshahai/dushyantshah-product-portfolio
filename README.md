## 🚀 Core Product: MorphEd

> `RAG` `EdTech` `LLM` `B2B SaaS`

### ⚡ Snapshot

* **What**: RAG-based MCQ generation platform
* **Who**: Indian colleges & coaching institutes
* **Impact**: 2 hours → <2 minutes content creation
* **Accuracy**: ~95% grounded outputs
* **Core edge**: Syllabus-aware hierarchical RAG (not generic LLM usage)

---

### Problem

Professors today rely on:

* Manual creation → time-intensive, low scalability
* Raw LLM usage → hallucinations, inconsistent difficulty
* Competitor tools → generic, not syllabus-grounded

---

### Solution

MorphEd ingests textbooks and builds a **4-level content hierarchy**, enabling:

* Context-grounded MCQ generation
* Difficulty-controlled outputs
* Instant assessment creation

---

### Beyond generation (closed learning loop)

* 📊 Batch-level analytics → class performance
* 🎯 Student-level insights → learning gaps
* 📍 Topic-level breakdown → targeted teaching

---

### My role

End-to-end ownership:

* Problem discovery
* Product strategy
* RAG architecture decisions
* UX flows
* Shipping

---

### Highlights

* Syllabus ingestion via **PDF / DOC / DOCX uploads (admin-driven)**
* Hierarchical curriculum mapping
* Difficulty-tuned MCQ generation
* On-demand + batch modes
